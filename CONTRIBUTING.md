# Contributing

Epsilon is one person's personal AI agent identity, not a reusable
brand. Read this before opening a PR.

## What's welcome

- Bug fixes: broken links, typos, asset rendering issues, missing
  alt text, accessibility regressions.
- Improvements that are genuinely broadly useful, like a clearer
  usage rule or a token entry that's underspecified.

## What's not

- Preference-based changes (different palette, different type,
  different tagline). The defaults are deliberate. If they don't fit
  your workflow, fork it and make it yours.
- Variants of the ε mark. One glyph, repeated everywhere.
- Re-use of "Epsilon", the avatar, or the ε mark to represent a
  different agent. Epsilon is the identity of `amcheste-ai-agent`
  and nothing else. See the README's "Not a shared logo" section.

## The bot-account pattern is the shareable part

The pattern (a personal AI agent that commits as a distinct GitHub
identity, with a human reviewer on every merge) is documented in
[`amcheste-ai-agent/epsilon-agent`](https://github.com/amcheste-ai-agent/epsilon-agent).
Adopt the pattern with your own brand.
