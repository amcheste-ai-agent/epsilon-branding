## [2026.05.0] - 2026-05
### Added
- Initial publication of the Epsilon brand system, the visual identity for the `amcheste-ai-agent` bot account.
- Avatar set: circle-crop PNGs at 80, 200, 460, and 1024 px plus the source SVG (avatar/).
- Social preview card: 1280 x 640 PNG and HTML source (social/).
- CLI splash banner: PNG and HTML source (cli/).
- Full interactive HTML branding kit (kit/Epsilon Branding Kit.html).
- Brand docs at docs/usage.md (do/don't rules for the ε mark) and docs/tokens.md (palette, type system, sizing).
- README badge stack (license, version, CAM family) tinted with Epsilon's honey amber `#F5B84A` for the license badge and shared Ink `#0B0B0C` for the structural badges.

- .github/CODEOWNERS routes PRs to @amcheste.
